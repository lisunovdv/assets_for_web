Нужно отслеживать успешную отправку формы, и передавать данные об этом событии в Facebook Pixel.
(Считаем, что код самого Facebook Pixel уже установлен на сайте).

Передача аналитики осуществляется вызовом метода fbq(), который принимает 2 основных параметра:

     fbq('track', 'Lead');

Также вызов данной функции может содержать Третий необязательный параметр в виде объекта, с помощью которого, например, можно передавать имя Лида, т.е.:

	fbq('track', 'Lead', {content_name:'zakaz_tovara'}); 

Полный список параметров можно посмотреть на Facebook for Developers: https://developers.facebook.com/docs/facebook-pixel/api-reference#parameters

Обычно вызов такой такой функции вешают на событие "submit" отправки формы, например, так:

			<form onsubmit="fbq('track', 'Lead');"> ...

или так, если используется jQuery:

		$('#form1').submit(function() { fbq('track', 'Lead'); });

При этом подразумеватеся, что валидация / проверка правильности ввода данных в форму осуществляется на фронтенеде
(Например, поля < input type=" email" required /> имеет атрибут required )
